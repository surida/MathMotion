부등이, 땅에 닿다

방정식 ax + b = 0 을 푼다는 건…
부등이가 미끄러져 땅(높이 0)에 닿는 지점을 찾는 거야!

↓ 땅을 찾아 내려가자 ↓
1막

방정식이 묻는 것 ax + b = 0 은 무슨 질문일까?

1
2x + 4 = 0
⬇ 함수로 변신!
y = 2x + 4
"= 0" 은 곧 "y(높이)가 0이 되는 x는?" 이라는 질문!
방정식의 좌변을 함수 y로 보면, "y=0일 때 x"를 찾는 문제가 됩니다.
2
높이 0은 그래프에서 땅(x축)이야!
y=0 인 곳 = x축. 그래서 "해 찾기 = 직선이 x축과 만나는 점 찾기".
2막

땅에 닿는 점 = 해 그래프가 알려주는 답

1
부등이가 직선을 따라 미끄러져… 땅에 닿았다!
2x+4=0 의 직선은 x=−2 에서 땅에 닿아요. 그 x가 바로 해!
2
검산해 볼까?
2 × (−2) + 4
= −4 + 4 = 0 ✓
해 −2를 넣으니 정말 0! 그래프가 맞았어 🎯
땅에 닿는 점의 x를 식에 넣으면 정확히 0. 그래프 = 방정식이 통합니다.
🎯
방정식 ax+b=0 의 해 = 그래프 y=ax+b 가 x축과 만나는 점(x절편). 그 값은 x = −b/a.
3막

땅 닿기 실험실 a·b 를 바꾸면 해가 어디로 미끄러질까?

2x + 4 = 0
해 x = −2직선이 x = −2 에서 땅에 닿아요
