부등이의 교점 사냥

포물선이 x축과 만나는 점 = 이차방정식의 근!
그래프와 방정식은 사실 같은 이야기예요.

↓ 두 세계를 잇자 ↓
1막

교점 = 근 그래프와 방정식은 한 몸

1
그래프가 x축과 만나면?
그 점에서 y = 0
ax2+bx+c = 0
→ 교점의 x좌표 = 방정식의 근
x축은 y=0인 곳! 거기서 만나니 식이 곧 =0 방정식이 돼요 🤝
그래프로 방정식을 풀 수도, 방정식으로 교점을 찾을 수도 있어요.
2
y = x2 − 2x − 3
⬇ x축 교점
(−1, 0) , (3, 0)
= (x+1)(x−3)=0 의 근 −1, 3
포물선이 x축을 뚫는 두 곳이 바로 방정식의 두 근! 인수분해 결과와 똑같죠 🎯
x²−2x−3=0 → (x+1)(x−3)=0 → x=−1 또는 3 = 교점의 x좌표.
2막

몇 번 만날까? 판별식 D = b²−4ac

1
D > 0 → x축과 두 점 (근 2개)
포물선이 x축을 가로지르면 두 점! 서로 다른 두 근이에요.
D=b²−4ac > 0 ⟺ 교점 2개 ⟺ 서로 다른 두 근.
2
D=0 접함(중근) · D<0 안 만남(근 없음)
x축에 살짝 닿으면 중근(1개), 아예 못 닿으면 실근 없음! 📏
위/아래로 볼록한 방향과 무관 — 포물선의 높이가 관건이에요.
🤝
x축 교점의 개수 = 이차방정식 근의 개수
🔎
판별식 D=b²−4ac → D>0 : 2개 · D=0 : 1개(중근) · D<0 : 0개
3막

가로선 스캔 실험실 y=k를 올리며 교점을 세어라!

k (높이)
0
포물선 y=x²−2x−3은 고정! 주황 가로선 y=k를 올리내리면 교점(=x²−2x−3=k의 해)이 변해요.
