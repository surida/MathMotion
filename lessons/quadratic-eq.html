부등이의 0의 비밀

(x+2)(x−3)=0 의 해는? x=−2 또는 3!
비밀은 "곱이 0이면 반드시 하나가 0" 이에요.

↓ 0의 비밀을 파헤치자 ↓
1막

곱이 0이면 하나가 0 0만 가진 특별한 힘

1
A × B = 6
→ 2×3? 1×6? 여러 경우
A × B = 0
→ A=0 또는 B=0 뿐!
곱이 6이면 짝이 여러 개지만, 곱이 0이면 반드시 하나가 0! 🔑
0은 특별해요 — 어떤 수든 0을 곱하면 0이고, 곱이 0이려면 한쪽이 0이어야 해요.
2
(x+2)(x−3) = 0
⬇ 곱이 0!
x+2=0 또는 x−3=0
두 식의 곱이 0 → x+2=0 또는 x−3=0!
A=(x+2), B=(x−3). AB=0이니 A=0 또는 B=0.
2막

그래서 두 근! 이차방정식 → 인수분해 → AB=0

1
x2−x−6=0
⬇ 인수분해
(x+2)(x−3)=0
⬇ AB=0
x=−2 또는 x=3
이차방정식을 인수분해해서 AB=0으로! → 두 근 −2, 3
x+2=0 → x=−2,   x−3=0 → x=3. (부호 주의!)
2
검산: (−2)²−(−2)−6 = 0 ✓
3²−3−6 = 0 ✓
그래프로 보면 포물선이
x축에 두 번 닿아요 (해 2개)
두 근 다 넣으면 0! 이차방정식은 보통 해가 2개예요 🎯
곱이 0이 되는 x = 포물선이 x축에 닿는 점 = 해.
🔑
AB=0 ⟺ A=0 또는 B=0. 이차방정식은 인수분해 후 각 인수=0으로 두 근을 구해요.
보너스

왜 포물선이 그려질까? 두 직선의 곱이라서!

y = (x+2)(x−3) = 두 직선 (x+2) 와 (x−3) 의 곱
⬆️
근 바깥(양 끝): 두 인수 부호가 같음 → 곱 + → 곡선이 위로
⬇️
두 근 사이: 두 인수 부호가 다름 → 곱 − → 곡선이 아래로
⚪
근에서: 한 인수가 0 → 곱 0 → x축에 닿음
가운데는 처지고 양끝은 올라가니 U자! 게다가 전개하면 x×x=x²(제곱) 때문에 직선이 아니라 휘어요 🪃
