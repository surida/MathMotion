# 학년·교과서 선택 + 확장형 메뉴 — 설계 문서

- **날짜**: 2026-06-12
- **범위**: 중2·중3 (중1 제외)
- **목표**: 진입 시 학년·교과서를 고르면 그 교과서 목차에 맞는 메뉴가 나오게. 출판사는 앞으로 계속 추가되므로 **데이터만 추가하면 되는 확장형 구조**.

## 핵심 결정
| 항목 | 결정 |
|---|---|
| 구조 | **레슨 풀(LESSONS) + 교과서별 매핑(CURRICULUM)** 분리. 같은 레슨을 여러 교과서가 공유 |
| 진입 | 시작 화면(학년→교과서) + localStorage 기억 + 상단바 칩으로 전환 |
| 확장 | 새 교과서 = `CURRICULUM[학년]`에 키 추가 / 새 레슨 = `LESSONS` 추가 후 id 참조. 코드 수정 0 |

## 데이터 구조
```js
const LESSONS = { '<id>': {emoji,title,desc,href}, ... };   // 개념 단위, 공유
const CURRICULUM = {
  '중2': { '교학사':[{n,name,lessons:[id,...]},...], '천재교육':[...] },
  '중3': { '동아출판사':[...] },
};
```

## 콘텐츠 매핑 (현재)
- **중2/교학사**(PDF 확정): 1.수와식→repeating-decimal / 2.부등식과방정식→ineq-mirror / 3.일차함수→linear-stairs,linear-equation / 4~6 준비중
- **중2/천재교육**(목차 잠정, 천재 PDF 오면 확정): 1.유리수와순환소수→repeating-decimal / 2.식의계산 / 3.부등식과연립방정식→ineq-mirror / 4.일차함수→linear-stairs,linear-equation / 5~7 준비중
- **중3/동아출판사**(PDF 확정): 실수와계산/다항식곱셈과인수분해/이차방정식/이차함수와그래프/삼각비/원의성질/통계 — 전부 준비중(중3 레슨 없음)

## 진입 흐름
1. load 시 localStorage(mm-grade, mm-pub) 확인 → 유효하면 바로 메뉴
2. 없으면 시작 화면: 학년 고르기 → 그 학년에 존재하는 교과서 고르기 → 저장 후 메뉴
3. 상단바 `[학년 ▾][교과서 ▾]` 칩 → 시작 화면 재오픈(전환)
- 단일 index.html 안에서 시작화면 ↔ 메뉴를 JS로 전환
- 교과서 목록은 선택 학년에 존재하는 것만 동적으로 표시

## 노트
- 기존 중2 단원은 교학사 PDF와 일치 → 교학사로 유지. 천재교육은 같은 레슨 공유, 목차는 PDF 도착 시 확정.
- 중1 제외(YAGNI). 추후 추가 시 CURRICULUM에 '중1' 키만 더하면 됨.
