- **날짜**: 2026-06-12
- **대상**: 중2 식의 계산 (교학사 "수와 식" / 천재 "식의 계산" 공유)

- **주제**: 지수법칙 (1) aᵐ×aⁿ=a^(m+n) (곱셈=지수 합), (2) (aᵐ)ⁿ=a^(mn) (거듭제곱=지수 곱)

## 핵심 결정
| 항목 | 결정 |
|---|---|
| 범위 | (1)+(2)만. "지수=곱한 횟수" 한 직관으로 합·곱 둘 다 |
| Motion | a 타일 세기 — 곱셈은 이어붙여 합(m+n), 거듭제곱은 격자로 묶어 곱(m×n) |
| 위치 | 교학사 수와 식 + 천재 식의 계산 (레슨 공유) |

## 구조 (3막)
```
[히어로]  a³×a²는 블록 몇 개? 지수=곱한 횟수
1막 웹툰  곱셈=합: a³(3블록)+a²(2블록) 이어붙이면 5 = a⁵
2막 웹툰  거듭제곱=곱: (a²)³ = 2개 묶음 3줄 = 2×3=6 = a⁶
3막 실험  a 블록 공장: [곱셈]/[거듭제곱] 모드, m·n 버튼 → 타일 합치기/격자, 식·개수 실시간
[마무리]  정리 + 퀴즈((a²)³ → a⁶ vs a⁵(더함) vs a⁸(2³ 혼동))
```

## 실험실
- 곱셈 모드: aᵐ(파랑 m) + aⁿ(빨강 n) 한 줄 → (m+n). `aᵐ×aⁿ=a^(m+n)`
- 거듭제곱 모드: aᵐ 묶음을 n줄 격자 → m×n. `(aᵐ)ⁿ=a^(mn)`
- m·n ± 버튼(곱셈 1~6, 거듭제곱 1~5), 타일 pop 애니메이션, 지수 계산 강조
- 같은 m,n이라도 곱셈은 한 줄(합), 거듭제곱은 격자(곱)로 대비
