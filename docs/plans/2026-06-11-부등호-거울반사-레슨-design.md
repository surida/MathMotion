# 부등호 거울 반사 레슨 — 설계 문서

- **날짜**: 2026-06-11
- **대상**: 중학교 1학년
- **주제**: 복잡한 부등호 연산의 기초 원리 — "양변에 더하거나 빼면 부등호는 그대로지만, 음수를 곱하거나 나누면 부등호가 뒤집힌다"를 **암기가 아니라 원리로** 이해시키기
- **형태**: 중학 수학 학습 사이트의 **첫 샘플 레슨 한 편**

## 핵심 결정 사항

| 항목 | 결정 | 이유 |
|---|---|---|
| 원리 비유 | **수직선 거울 반사** (0을 거울축으로) | 수학적으로 가장 정확하고 애니메이션으로 강력하게 보임 |
| 구현 | **단일 HTML 파일** (SVG + CSS/JS, 의존성 0) | 빌드·설치 없이 더블클릭 실행, 공유·배포 쉬움, 콘텐츠 수명 김 |
| 캐릭터/서사 | **부등호 악어 '부등이' + 거울 동굴** | 비유(거울 반사)와 이야기(동굴)가 한 몸이 되도록 |
| 인터랙션 | **슬라이더 + 연산 버튼** | 0을 부드럽게 통과시키며 "뒤집히는 순간"을 몸으로 체감 |

## 핵심 교육 원리

음수를 곱한다 = **0을 기준으로 거울에 비춘다 (반사)** → 수의 순서가 뒤집힌다 → 부등호도 뒤집힌다.
거울축(0)을 항상 화면에 점선으로 표시해 "곱하기 음수 = 반사"를 시각적으로 등치시키는 것이 레슨 전체의 핵심.

## 전체 구조 — 위→아래 스크롤, 3막 구성

```
[히어로]  부등이 등장 + 제목 "부등호가 뒤집힌다고?!"
1막 웹툰  평화로운 부등 마을 — 덧셈·뺄셈: 부등호 안 변함
2막 웹툰  거울 동굴의 비밀 — 음수 곱하기: 좌우 반전 → 입 방향 뒤집힘 ("왜?"를 거울 반사로)
3막 실험  인터랙티브 실험실 — 슬라이더로 직접 뒤집어보기
[마무리]  한 줄 정리 + 퀴즈 1문제
```

설계 의도: 1막에서 "당연한 것"으로 기준선을 만들고 → 2막에서 "왜 이것만 다르지?"라는 인지적 긴장을 일으킨 뒤 → 3막에서 직접 해결. 답을 먼저 주지 않고 궁금하게 만든 다음 풀게 한다.

## 웹툰 상세

**캐릭터 (모두 인라인 SVG, 이미지 파일 없음)**
- **부등이**: 초록 악어. 벌린 입이 곧 부등호 `<`. 항상 큰 수 쪽으로 입을 벌림. 눈·볼터치로 귀엽게.
- **숫자 친구들**: 동글동글한 수 캐릭터(예: 2와 5), 표정 있음.
- **거울 동굴**: 반짝이는 거울 벽, 들어가면 좌우 반전.

**1막 — 평화로운 부등 마을 (덧셈·뺄셈), 컷 4개**
1. `2 < 5`, 부등이가 5쪽으로 입 벌림. "난 항상 큰 쪽을 향해!"
2. 양쪽에 같은 추 `+3` → `5 < 8`. 점들이 나란히 오른쪽으로 미끄러짐.
3. 입 방향 그대로. "둘 다 똑같이 움직이니 순서는 안 바뀌지~"
4. 정리: "양변에 같은 수를 더하거나 빼면 → 부등호 그대로 ✅"

**2막 — 거울 동굴의 비밀 (음수 곱하기), 컷 5개**
1. `2 < 5`, 평화로움.
2. "×(−1) 동굴"에 두 숫자가 들어감. 거울 벽 반짝.
3. **결정적 컷**: 0을 거울축으로 점들이 휙 반사 → `-2`, `-5`. 작던 2가 오른쪽(−2, 더 큼), 크던 5가 왼쪽(−5, 더 작음)으로 자리가 바뀜.
4. 부등이도 거울상이 되어 입 방향 반대로. `-2 > -5`. "헉, 내 입이 반대로?!"
5. 정리: "음수를 곱하면 = 거울로 비추기 → 순서가 뒤집힘 → 부등호도 뒤집힘 🔄"

3번 컷이 레슨 전체의 '아하 순간'. 거울축(0)을 점선으로 항상 표시.

## 인터랙티브 실험실 상세

화면 중앙 **수직선**, 그 위 두 점 **A(파랑)**·**B(빨강)**. 0 위치에 **거울축 점선** 상시 표시.

```
   A=2        B=5
────●──────────●────────►
    [거울축:0]
   "2 < 5"  ← 현재 부등식 (악어 입 방향으로 표시)
```

**조작부 (수직선 아래)**
- 연산 선택 버튼: `+` `−` `×` `÷` (한 번에 하나 활성)
- 슬라이더: −5 ↔ +5, 가운데 0. 끌면 양변에 실시간 적용.
- 슬라이더 옆 현재 값 표시: 예 `× (−1.0)`

**핵심 동작 — ×에서 슬라이더를 양수→0→음수로 끌 때**
1. 양수 구간: 두 점이 늘었다 줄었다 하지만 순서 유지, 부등호 그대로.
2. 0 통과: 두 점이 0으로 모임(=), 부등호 잠깐 `=`.
3. 음수 구간: 두 점이 거울축을 넘어 반대편으로. A가 B 오른쪽으로 가는 순간 악어 입이 회전 애니메이션으로 뒤집히고, 라벨이 `2 < 5` → `-2 > -5`로 바뀜.

**피드백 장치**
- 뒤집히는 순간: 화면 살짝 번쩍 + "🔄 뒤집혔다!" 토스트.
- 상단 **"왜?"** 토글 → 각 점에서 0까지 거리 화살표(거울 반사 보조선) 표시.
- 프리셋 버튼: "×(−1) 해보기", "+3 해보기".

설계 의도: 슬라이더가 0을 연속으로 통과하게 해, "뒤집힘은 0을 경계로 일어난다"는 구조를 학생이 스스로 발견하게 함. 음수가 '특별해서'가 아니라 '거울 반대편이라서'임을 체감.

## 비주얼·기술 구현

**비주얼 톤**
- 밝고 둥근 파스텔(악어 초록 + 하늘/크림), 큰 라운드 코너, 손글씨 느낌 제목.
- 모바일 우선 세로 스크롤. 글자·버튼 크게.
- 한글 전체, 중1 눈높이 어휘.

**기술 (단일 `index.html`)**
- 시맨틱 HTML 막별 `<section>` + 내부 `<style>` + `<script>`. 외부 의존성 0.
- 캐릭터·수직선: 인라인 SVG. 점·악어 입은 JS로 좌표/회전 변경.
- 애니메이션: CSS transition + JS 속성 갱신.
- 스크롤 등장: `IntersectionObserver` 페이드인.
- 접근성: 버튼 키보드 포커스, 슬라이더 `aria-label`, 색 대비 확보.

**파일 구조**
```
MathMotion/
├── index.html          ← 샘플 레슨 한 편 (전부 여기)
└── docs/plans/
    └── 2026-06-11-부등호-거울반사-레슨-design.md
```
