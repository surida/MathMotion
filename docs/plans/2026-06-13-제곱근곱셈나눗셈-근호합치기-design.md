# 제곱근의 곱셈과 나눗셈 레슨 — 설계 문서

- **날짜**: 2026-06-13
- **대상**: 중3 수와 연산 / 근호를 포함한 식의 계산
- **참고**: 천재 이준열 중3 "01 수와 연산" PDF (2.근호를 포함한 식의 곱셈과 나눗셈)
- **주제**: √a×√b=√(ab), √a÷√b=√(a/b) — 근호 안 수끼리 곱하고 나눈다
- **형태**: 단일 `lessons/sqrt-mul-div.html` (제곱근 레슨 후속)

## 핵심 결정
| 항목 | 결정 |
|---|---|
| 핵심 원리 | 근호 안 수끼리 곱/나눗. (√a√b)²=ab 라서 √(ab) |
| Motion | 근호 합치기 — 두 근호의 속 수가 하나의 근호로 합쳐지고, 완전제곱이면 정수/아니면 간단히(a√b) |
| 범위 | 곱셈·나눗셈 집중 (분모 유리화·근호 밖 빼기는 맛보기만). 심플 |
| 위치 | 천재 근호를 포함한 식의 계산(n2) + 동아 실수와 그 계산(n1) 공유 |

## 구조 (3막)
```
[히어로]  √2×√3=√6, √6÷√2=√3 — 근호 안 수끼리!
1막 웹툰  곱셈 √a×√b=√(ab). 왜? (√2√3)²=6 → √6. √2×√8=√16=4
2막 웹툰  나눗셈 √a÷√b=√(a/b). √12÷√3=√4=2. 간단히 √50=5√2
[법칙]    √a×√b=√(ab) / √a÷√b=√(a/b)
3막 실험  근호 합치기 기계: [곱셈×]/[나눗셈÷] + 예시 버튼 → 두 근호가 √(ab)/√(a/b)로 합쳐지고 간단히(정수 or a√b)
[마무리]  퀴즈 2 (√3×√12=6, √20÷√5=2)
```

## 실험실
- 모드 ×/÷ 토글 → 각 모드의 예시 버튼 세트
- 예시 클릭 → 3단계 reveal: ① √a op √b ② = √(a op b) = √(inner) ③ = 간단히(정수/ a√b). note로 설명
- simplify(n): 완전제곱 인수 추출 → {coef,rad}. 모든 예시 결과 node 검산 통과
- 모든 갱신 지정 요소(#line1~3,#note)로만, 등장 애니메이션은 opacity transition만(쉬는 상태 항상 보임)
