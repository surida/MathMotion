# 제곱근 레슨 — 설계 문서

- **날짜**: 2026-06-13
- **대상**: 중3 수와 연산 (천재 제곱근과 실수 / 동아 실수와 그 계산 공유)
- **참고**: 천재 이준열 중3 "01 수와 연산" PDF (제곱근의 뜻·성질·대소, 무리수·실수)
- **주제**: 제곱근의 뜻 = 넓이가 a인 정사각형의 한 변(√a) = 제곱의 반대
- **형태**: 단일 `lessons/square-root.html`

## 핵심 결정
| 항목 | 결정 |
|---|---|
| 핵심 원리 | √a = 넓이 a 정사각형의 한 변. 제곱(변→넓이)의 역연산 |
| Motion | 넓이↔변 양방향 정사각형 ([제곱근:넓이→변] / [제곱:변→넓이] 모드) |
| 위치 | 천재 중3 제곱근과 실수(n1) + 동아 중3 실수와 그 계산(n1) 공유 |

## 구조 (3막)
```
[히어로]  넓이 9 → 한 변 3, 넓이 2 → 한 변 √2
1막 웹툰  제곱(변 3→넓이 9) vs 제곱근(넓이 9→변 3 = √9). 제곱근은 제곱의 반대
2막 웹툰  x²=9 → x=3,−3 (양의 제곱근 √9=3, 음의 제곱근 −3) / 넓이 2 → √2 무리수
[법칙]    a의 제곱근 = ±√a (a>0). 완전제곱수면 유리수, 아니면 무리수
3막 실험  넓이↔변 정사각형: 모드 토글, 넓이/변 버튼 → 정사각형 실시간 리사이즈 + 변=√넓이(유리/무리 배지) + ±√a 안내
[마무리]  퀴즈 2 (넓이 16→한 변 4 / 9의 제곱근 ±3)
```

## 실험실
- 제곱근 모드: 넓이 [−][+](1~16) → 정사각형 width/height = √넓이·UNIT (CSS transition), 한 변=√a, 완전제곱(1,4,9,16)→유리수 배지/그 외→무리수 배지, "±√a 둘" 안내
- 제곱 모드: 한 변 [−][+](1~6) → 넓이=변²
- 모든 갱신은 지정 요소(#sq,#eqBig 등)로만, 등장 애니메이션 없이 transition만 (쉬는 상태 항상 보임)

## 노트
- 완전제곱 vs 아닌 경우를 같은 정사각형에서 보여 "유리/무리" 직관 → 다음 무리수 단원 연결
- 무리수·실수·수직선 작도는 후속 레슨 (YAGNI)
