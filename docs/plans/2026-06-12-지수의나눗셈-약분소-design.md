# 지수의 나눗셈 (지수법칙 3) 레슨 — 설계 문서

- **날짜**: 2026-06-12
- **대상**: 중2 식의 계산 (교학사 수와 식 / 천재 식의 계산 공유)
- **참고**: 교학사 1단원 수와 식 PDF, 천재 류희찬 02 식의 계산 PDF
- **주제**: aᵐ÷aⁿ (a≠0) — 약분으로 지수를 뺀다
- **형태**: 단일 `lessons/exponent-division.html` (곱셈 레슨 블록 재활용)

## 핵심 결정
| 항목 | 결정 |
|---|---|
| 원리 | 나눗셈=분수=약분. 위아래 a를 짝 지우면 남는 쪽이 답 |
| Motion | 분수 aᵐ/aⁿ에서 짝(min(m,n)쌍) 타일이 펑 사라지고 남은 타일이 결과 |
| 세 경우 | m>n→a^(m−n) / m=n→1 / m<n→1/a^(n−m) (중2는 분수 표기, 음수지수 X) |
| 위치 | 교학사 수와 식 + 천재 식의 계산 (exponent-laws 옆) |

## 구조 (3막)
```
[히어로]  a⁵÷a²? 나눗셈=분수=약분
1막 웹툰  a⁵/a² → 짝 2쌍 지우면 위 3개 = a³ (지수 5−2)
2막 웹툰  m=n→다 지워져 1 / m<n→아래 남아 1/a^(n−m)
3막 실험  a 약분소: aᵐ/aⁿ, m·n 버튼 → 짝 타일 cancel 애니메이션 후 남은 쪽 결과
[마무리]  퀴즈 2: a⁵÷a²(a³/a⁷/a^2.5), a²÷a⁵(1/a³/a³/1/a⁷)
```

## 실험실 동작
- 2단계 렌더: ① 위 m·아래 n 타일, 첫 k=min(m,n)쌍에 빨간 슬래시(matched) → ② 0.7s 후 cancel(scale 0) → ③ 1.18s 후 남은 쪽으로 재구성(분수바 표시/숨김) + 결과식
- m>n: 위 타일만 남고 바 숨김(a^(m−n)) / m<n: 위 "1" + 바 + 아래 타일(1/a^(n−m)) / m=n: "1"
- 범위 m,n 1~6

## 노트
- 곱셈은 "이어 곱하기(×)→합", 나눗셈은 "약분(짝 지우기)→차" 대칭 구도
- 음수 지수 미사용(중2). m<n은 1/a^(n−m) 분수로
- (4) (ab)ⁿ·(a/b)ⁿ 분배는 다음 레슨(YAGNI)
