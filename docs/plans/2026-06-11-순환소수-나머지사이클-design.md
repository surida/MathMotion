# 순환소수의 원리 레슨 — 설계 문서

- **날짜**: 2026-06-11
- **대상**: 중학교 2학년 (1단원 수와 식 / 유리수와 순환소수)
- **주제**: 순환소수의 원리 — "분수를 나눗셈하면 나머지의 종류가 유한(0~b−1)해서, 0에 닿으면 유한소수, 0이 아닌 나머지가 다시 나오면 순환소수"
- **형태**: 단일 `lessons/repeating-decimal.html`

## 핵심 결정
| 항목 | 결정 |
|---|---|
| 핵심 원리 | 나눗셈의 **나머지 사이클** (비둘기집 원리). 유한/순환은 한 원리의 두 결과 |
| Motion | 분수 선택 → '나눗셈 재생' → 나머지 토큰이 **0~(b−1) 원형 링**을 돎. 0칸=유한, 같은 칸 재방문=순환마디 |
| 위치 | 중2 1단원 수와 식 (첫 레슨) |

## 구조 (3막)
```
[히어로]  1÷3=0.333…, 3÷7=0.428571… 어떤 건 끝나고 어떤 건 반복? 비밀은 나머지!
1막 웹툰  나머지 관찰: 1/4는 나머지 0→끝(유한) vs 1/3은 나머지 1 반복→안 끝남(순환)
2막 웹툰  7로 나누면 나머지는 0~6, 7칸뿐! 칸이 유한 → 0 도달(끝) 또는 같은 칸 재방문(반복). 비둘기집
[법칙]    유한 ⟺ 나머지 0 / 순환 ⟺ 나머지 반복
3막 실험  나눗셈 회전판: 분수 고르면 단계 재생, 나머지 토큰이 링을 돎, 전광판에 몫, 순환마디 자동 표시
[마무리]  정리 + 퀴즈(7로 나눠 0이 안 나오면? → 순환)
```

## 실험실 디테일
- 분모 b개의 칸을 원형 링(0..b−1)으로. 토큰(빨간 마커)이 현재 나머지 칸에 위치, 단계마다 hop
- 한 단계 = `나머지×10 → 몫 1자리(전광판 LED) + 새 나머지`. 나레이션으로 계산 설명
- 0칸 도달=초록(끝/유한), 0 아닌 칸 재방문=빨강 구간(순환마디) + 토큰이 사이클을 계속 회전
- 프리셋: 1/4·2/5(유한) vs 1/3·1/6·3/7·5/11(순환). '다시 재생' 버튼
- 알고리즘: `divisionSteps(num,den)` — seen 딕셔너리로 나머지 사이클 탐지, prefix+한 주기 반환

## 의도적 제외 (YAGNI)
- ❌ 임의 분수 직접 입력(분모 큰 링 방지) — 프리셋으로 b≤11 제한
- ❌ 유한소수 판별(분모 2·5) → 다음 레슨(B안)
- ❌ 순환소수→분수 변환 → 후속 레슨(D안)
