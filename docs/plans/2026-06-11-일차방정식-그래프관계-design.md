# 일차함수와 일차방정식의 관계 레슨 — 설계 문서

- **날짜**: 2026-06-11
- **대상**: 중학교 2학년 (3단원 일차함수, 계단 레슨의 후속)
- **주제**: 일차함수와 일차방정식의 관계 — "방정식 `ax+b=0`을 푼다 = 그래프 `y=ax+b`가 x축(땅, 높이 0)과 만나는 점을 찾는다"
- **형태**: 단일 `lessons/linear-equation.html` (SVG+CSS+JS, 의존성 0)

## 핵심 결정

| 항목 | 결정 |
|---|---|
| 핵심 관계 | `ax+b=0`의 해 = 그래프가 **x축과 만나는 점**(x절편) = `x=-b/a` |
| Motion | a·b ± 버튼 → 직선 이동, **땅 닿는 점이 빛나고 부등이가 미끄러짐**, 해 실시간 |
| 범위 | `ax+b=0` 집중 (a≠0; a=0은 "해 없음" 안내로 자연 처리) |
| 마스코트/비유 | 부등이 — 계단 레슨의 직선 위 캐릭터가 "땅(y=0)에 닿는다" |

## 교육 관점
"방정식 = 함수의 한 장면(y=0인 순간)". 이 관점은 이후 이차방정식·연립방정식("그래프가 만나는 점")으로 확장되는 큰 그림의 첫 단추.

## 구조 (3막)
```
[히어로]  부등이가 직선 위에서 "내가 땅에 닿는 곳은?"
1막 웹툰  방정식이 묻는 것 = "y(높이)=0인 x는?" → 방정식을 함수로 변신, y=0은 x축(땅)
2막 웹툰  땅에 닿는 점 = 해. 예) 2x+4=0 → x=-2. 검산 2×(-2)+4=0 ✓
[법칙]    해 = x절편 = -b/a
3막 실험  땅 닿기 실험실: a·b 버튼 → 직선 이동, x절편 빛남+부등이 미끄러짐, 해 -b/a 실시간
[마무리]  정리 + 퀴즈 (3x-6=0 → x=2)
```

## 실험실 디테일
- 좌표평면(계단 레슨 그래프 재사용), **x축을 "땅"으로 강조**(갈색 지면 + 흙 영역 + "땅(y=0)" 라벨)
- 직선 `y=ax+b`, x절편에 **빛나는 해 점(halo 펄스)** + 부등이가 그 점으로 슬라이드
- 상단: `a x + b = 0` / **해 x = −b/a** (기약분수 표시, 예 `-2/3`)
- 레이어 분리: 격자(고정) / 직선·검산(매번 그림) / 해 점·부등이(영구, transition으로 미끄러짐)
- **"왜? 검산 보기"** 토글: "여기선 높이=0 ✓" + y축의 b 강조
- a=0: "땅과 평행 → 해 없음"(b=0이면 "모든 수가 해") 안내, 해 점·부등이 숨김
- 범위: a∈[-3,3], b∈[-6,6]

## 의도적 제외 (YAGNI)
- ❌ ax+b=c 일반화, 두 직선 교점(연립) → 다음 레슨
- ❌ 특수해 깊은 탐구(a=0)는 안내 문구로만
