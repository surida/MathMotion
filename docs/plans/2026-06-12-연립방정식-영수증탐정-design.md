# 연립일차방정식(가감법) 레슨 — 설계 문서

- **날짜**: 2026-06-12
- **대상**: 중2 (교학사 부등식과 방정식 / 천재 일차부등식과 연립일차방정식 공유)
- **참고**: 교학사 2단원 PDF, 천재 03 일차부등식과 연립일차방정식 PDF
- **주제**: 연립일차방정식 = 미지수 하나를 없애서(소거) 푼다 — 가감법 중심
- **형태**: 단일 `lessons/simultaneous-equations.html`

## 핵심 결정
| 항목 | 결정 |
|---|---|
| 핵심 원리 | 미지수 2개 → 식도 2개 필요. 두 식을 더하거나 빼서 **한 미지수 소거** |
| Motion | **영수증 빼기** — 공통 항목(같은 항)이 상쇄되어 한 미지수가 사라짐(가감법) |
| 범위 | 가감법 중심, 대입은 "구한 값 넣기"로 간단히. (대입법 단독은 생략 — 심플) |
| 위치 | 교학사 부등식과 방정식 + 천재 일차부등식과 연립일차방정식 |

## 구조 (3막)
```
[히어로]  영수증 두 장으로 연필·지우개 값 알아내기
1막 웹툰  영수증 1장 → 가능성 여러 개 / 2장 → 딱 하나 (미지수 2개엔 식 2개)
2막 웹툰  가감법: [2x+y=1500] − [x+y=900] → y 상쇄 → x=600, 대입 → y=300
[법칙]    가감법: 더하거나 빼서 한 미지수 소거 → 남은 식 풀고 대입
3막 실험  영수증 탐정: 연립식 프리셋 + [①−②]/[①+②] → 소거되는 미지수 빗금 → 해 (x,y) + 검산
[마무리]  정리(같은 계수=빼서, 부호 반대=더해서 소거) + 퀴즈(x+y=5, x−y=1 → y 소거는 +)
```

## 실험실 동작 (교과서 가감법 세로셈)
- 프리셋: 연필·지우개(2x+y=1500, x+y=900 → −로 y소거), x+y=4/x−y=2(+ 또는 −), 3x+2y=12/x+2y=8(−로 y소거)
- [①−②]/[①+②] 선택 → 계수 합/차 계산 → 0이 되는 미지수 소거(빗금) → 남은 일차방정식 풀이 → ②에 대입 → 해 + 검산 ✓
- 소거 안 되는 연산 선택 시 "이 연산으론 안 사라져요" 안내 (＋/− 선택의 의미 학습)
- 모든 계산 node로 검산 통과, 렌더 헤드리스 확인

## 노트
- "영수증 1장=여러 가능성, 2장=하나"로 미지수=식 개수 직관 먼저
- 가감법 시각 = '짝 지우기(약분)' 비주얼 재활용
- 대입법 단독 레슨/계수맞추기(×k)는 다음 확장 (YAGNI)
