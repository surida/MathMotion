# MathMotion 벤치마킹 리포트 (종합)

작성일: 2026-06-18 · 범위: 교육 · 기능 · 시장

## 0. 요약 (Executive Summary)

MathMotion의 정체성은 **① 웹툰 스토리텔링 + ② 인터랙티브 "Motion"(직접 조작) + ③ 한국
교육과정·교과서 매핑 + ④ 교사용 오개념 대시보드 + ⑤ 무료·웹**의 결합이다. 조사한 어떤
경쟁 서비스도 이 다섯을 동시에 갖추지 못한다.

- 가장 가까운 단일 경쟁자는 **Mathigon**(인터랙티브 내러티브, 무료)이나, 영어 중심이고
  한국 교과서에 매핑되지 않으며 웹툰 정서·교사 오개념 분석이 없다.
- 한국 **학습만화**(수학도둑·판타지 수학대전 등)는 스토리는 강하나 **종이·정적**이라
  상호작용·기록·즉시 피드백이 없다.
- 한국 앱(**콴다·수학대왕·노리**)은 AI 풀이검색·개인화에 집중, "개념을 이야기와 움직임으로
  이해"라는 축이 아니다.

→ MathMotion은 **"한국 교과서 기반 인터랙티브 웹툰 + 교사 오개념 대시보드"**라는 비어 있는
교차점에 위치한다. 기회가 크지만, 콘텐츠 양·개인화·모바일·게이미피케이션은 보강 과제다.

## 1. 경쟁 지형 (5개 카테고리)

| 카테고리 | 대표 서비스 | 한 줄 정의 |

| 학습만화 | 수학도둑, 판타지 수학대전, 수학 비밀일기 | 만화 스토리로 개념·흥미 (종이책) |

## 3. MathMotion의 강점

1. **유일한 조합**: "교과서 단원 → 웹툰 → Motion → 단계별 퀴즈 → 교사 오개념 분석"을 한 흐름에.
2. **교사 인사이트**: 점수가 아닌 **오개념 분포**(어떤 오답을 골랐나)를 학급 단위로 제공 —
   콴다/수학대왕(개인 학습)·학습만화(피드백 없음)에 없는 축. Desmos·노리에 견줄 교실 가치.
3. **한국 교과서 정합성**: 학년·출판사·단원에 정확히 매핑(공교육 교사 도입 친화) — 글로벌
   서비스(Mathigon·Brilliant)가 못 따라오는 로컬 강점.
4. **즉시성·무접속 마찰**: 설치·결제 없는 무료 웹, 학급 코드만으로 입장.
5. **단계별 해금(Lv1→Lv3)**: 빠른 학생 도전 + 약한 학생 보호 — Brilliant식 점증 난이도를 한국형으로.

## 4. 약점 · 갭 (경쟁 대비)

| 갭 | 누가 잘하나 | MathMotion 현황 |
|---|---|---|
| 콘텐츠 양 | 콴다(1,000만 문제), Mathigon(전 학년) | 중2~중3 일부 단원 |
| 적응형 개인화 | 노리, Brilliant | 단계 해금은 있으나 "약점 자동 추천"은 없음 |
| 조작 도구 라이브러리 | Mathigon(Polypad) | 레슨마다 개별 제작(재사용 도구셋 없음) |
| 게이미피케이션 | Prodigy, 똑똑수학탐험대 | 뱃지·수집·연속학습 없음 |
| 모바일 앱 | 대부분 | 웹만(반응형) |
| 주관식·풀이 입력 | 콴다, 노리 | 객관식 중심 |
| 다국어 | Mathigon(20+) | 한국어 only |

## 5. 로드맵 제언 (벤치마킹에서 가져올 것)

- **단기(차별화 강화)**
  - admin 고도화: 미응시 학생·**레벨 도달률**·반 오개념 Top3 (Desmos/노리의 교실 가치 흡수).
  - 가벼운 게이미피케이션: 레슨 완주 뱃지·연속 학습(스트릭) — Prodigy/똑똑수학탐험대식 동기, 단 "수학=통행료" 함정은 피함.
- **중기(범위 확장)**
  - 단원 커버리지 확대(중1·고1) + 단원 간 선수학습 연결.
  - **재사용 Motion 컴포넌트**(슬라이더 그래프·대수타일·수직선)로 제작 속도↑ — Mathigon Polypad 발상.
- **장기(개인화·접근성)**
  - 오답 데이터를 활용한 **약점 자동 추천**(노리·Brilliant 모델, 이미 attempts 데이터 보유 → 강점).
  - PWA로 모바일 설치·오프라인.
- **하지 않을 것(YAGNI)**: AI 풀이검색(콴다 레드오션), 방대한 문제 DB 경쟁 — MathMotion은 "이해" 포지션 유지.

## 6. 포지셔닝 한 줄

> **"한국 교과서를 웹툰과 직접 조작으로 이해시키고, 교사에겐 오개념을 보여주는 무료 학습 사이트."**
> 학습만화의 재미 + Mathigon의 상호작용 + 노리의 교실 인사이트를, 한국 공교육 맥락에서 하나로.
